Class Deny

Use this policy to deny grant against roles and scope. ie. new Deny({ roles: ["user", "admin"], scope: ["read", "write"] }) You should always define policies while creating your model schema Should be used within a Policies instance. Use the grant method in order to grant against a provided set of roles, scope and effect
